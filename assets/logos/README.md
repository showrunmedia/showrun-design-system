## When to use which

| Variant | Use for |
|---|---|
| Horizontal A | DEFAULT for most applications — documents, decks, email signatures |
| Horizontal B | Alternate horizontal lockup — different text/icon arrangement |
| Stacked A | Square / vertical spaces — social profile images, app icons, tight layouts |
| Stacked B | Alternate stacked lockup |
| Icon | Favicon, small-format, watermarks, when the name appears elsewhere in context |

For Showrun OS specifically:
| Variant | Use for |
|---|---|
| Showrun OS Horizontal A/B | Product pages, OS-specific decks, client onboarding |
| Showrun OS Stack | Square contexts for the OS product |

## Rules (from showrun-brand-guidelines skill)

- **Minimum clear space:** the height of the "S" in Showrun on all sides
- **Never** stretch, rotate, recolor, or add effects (drop shadows, outlines, glows)
- **Dark backgrounds** → use light/reversed variant
- **Light backgrounds** → use dark/standard variant
- **On photography** → place on an area with sufficient contrast, or add a subtle overlay
- **Icon alone** → only when "Showrun" name appears elsewhere in context
- **Minimum size:** Icon 24px, Horizontal 120px wide, Stacked 80px wide
